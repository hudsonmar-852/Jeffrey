# Jeffrey Wellness OS v4 — Sprint 1 Foundation

This directory establishes the production foundation for the future Jeffrey Wellness OS without replacing the existing Daily Reminder application.

## Sprint 1 scope

- Brand positioning and voice
- Design tokens
- UX principles
- Information architecture
- AI content and safety rules
- Component inventory
- Production roadmap handoff

## Core promise

**One minute. Better every day.**

Jeffrey Wellness OS is a human-first, evidence-informed wellness engagement experience. It helps busy people take one small, realistic action each day, unlock useful guidance, and privately ask Jeffrey for support without sales pressure.

## Current status

Sprint 1 foundation only. No existing production route is replaced.
